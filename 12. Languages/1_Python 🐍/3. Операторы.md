
| **Арифметика**           |                            |                                           |                                |                                                              |
| ------------------------ | -------------------------- | ----------------------------------------- | ------------------------------ | ------------------------------------------------------------ |
| `+`                      | Сложение                   | `3 + 2` → `5`                             | Складывает числа               | _«Как на калькуляторе: 3 яблока + 2 яблока = 5»_             |
| `-`                      | Вычитание                  | `5 - 2` → `3`                             | Вычитает                       | _«Отдал 2 яблока — осталось 3»_                              |
| `*`                      | Умножение                  | `4 * 3` → `12`                            | Умножает                       | _«4 коробки по 3 яблока = 12»_                               |
| `/`                      | Деление                    | `7 / 2` → `3.5`                           | Делит, всегда даёт `float`     | _«7 яблок на 2 человек = по 3.5»_                            |
| `//`                     | Целочисленное деление      | `7 // 2` → `3`                            | Делит и отбрасывает дробь      | _«Сколько _целых_ пар можно сделать из 7 носков? 3»_         |
| `%`                      | Остаток от деления         | `7 % 2` → `1`                             | Сколько осталось?              | _«7 носков → 3 пары, и 1 носок без пары»_                    |
| `**`                     | Возведение в степень       | `2 ** 3` → `8`                            | 2×2×2                          | _«2 в кубе — как кубик 2×2×2»_                               |
| **Сравнение**            |                            |                                           |                                |                                                              |
| `==`                     | Равно?                     | `5 == 5` → `True`                         | Сравнивает значения            | _«Это ОДИНАКОВЫЕ коробки?» (не путать с `=` — это _вопрос_)_ |
| `!=`                     | Не равно?                  | `5 != 3` → `True`                         | Не равны?                      | _«Разные? Тогда да!»_                                        |
| `>` / `<`                | Больше / Меньше            | `5 > 3` → `True`                          | Сравнение                      | _«5 лет старше 3-х? Да»_                                     |
| `>=` / `<=`              | Больше/равно, Меньше/равно | `age >= 18`                               |                                | _«Можно в кино, если 18 или больше»_                         |
| **Логика**               |                            |                                           |                                |                                                              |
| `and`                    | И                          | `x > 0 and x < 10`                        | Оба условия — истина           | _«Если дождь И холодно → зонт И куртка»_                     |
| `or`                     | Или                        | `weather == "дождь" or weather == "снег"` | Хоть одно — истина             | _«Если дождь ИЛИ снег → зонт»_                               |
| `not`                    | Не                         | `not is_raining`                          | Отрицание                      | _«Если НЕ дождь → можно без зонта»_                          |
| **Присваивание**         |                            |                                           |                                |                                                              |
| `=`                      | Присвоить                  | `x = 10`                                  | Положить значение в переменную | _«Положи 10 в коробку `x`»_                                  |
| `+=`                     | Увеличить и присвоить      | `x += 2` → `x = x + 2`                    | Быстрее писать                 | _«Добавь 2 к тому, что уже есть»_                            |
| `-=`                     | Уменьшить и присвоить      | `count -= 1`                              |                                | _«Убери одну тарелку»_                                       |
| **Строки**               |                            |                                           |                                |                                                              |
| `+`                      | Склейка                    | `"Привет, " + name`                       | Соединяет строки               | _«Склеить два листочка: "Привет" + "Аня" = "Привет, Аня"»_   |
| `*`                      | Повторение                 | `"О!" * 3` → `"О!О!О!"`                   | Повторить N раз                | _«Крикни "О!" три раза подряд»_                              |
| **Продвинутое**          |                            |                                           |                                |                                                              |
| `in`                     | Принадлежит?               | `"а" in "мама"` → `True`                  | Есть ли элемент?               | _«Есть ли буква "а" в слове "мама"?»_                        |
| `is`                     | Тот же объект?             | `a is b`                                  |                                | _«Это ОДНА и та же коробка? (не путать с `==`!)»_            |
