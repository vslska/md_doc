
### Синтаксис регулярных выражений, используемый командами `grep`, `awk` и `sed`


| Соответсвие                                                        | Используемое выражение                            | Пример                                                                                                                |
| ------------------------------------------------------------------ | ------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Начало строки                                                      | `^ (carret)`                                      | `^`a = строка, *начинающаяся* с `a`                                                                                   |
| Конец строки                                                       | `$ (dollar)`                                      | !`$` = строка, *заканчивающаяся* `восклицательным` знаком                                                             |
| Любой `одиночный символ`<br>(кроме новой строки)                   | `. (dot)`                                         | `…` = любые три последовательных<br>символа                                                                           |
| Знаки вставки, `доллара` или <br>`любой другой` специальный символ | `\c`                                              | $=знак доллара                                                                                                        |
| *Ноль* или *более* вхождений<br>выражения `E`                      | `E*`                                              | `_*` = *ноль* или *более* знаков подчер-<br>кивания                                                                   |
| `Любой одиночный` символ в наборе                                  | `[characters]`                                    | [aeiouAEIOU] = любая гласная                                                                                          |
| `Любой одиночный` символ, <br>*не* входящий в набор                | `[^characters]`                                   | `[^aeiouAEIOU]` = любая не гласная                                                                                    |
| `Любой символ` в диапазоне<br>между `с1` и `с2`                    | `[c1-c2]`                                         | [0-9] = любая цифра                                                                                                   |
| `Любой символ` *вне* диапазона<br>между `c1` и `c2`                | `[^c1-c2]`                                        | `[^0-9]` = любой нецифровой символ                                                                                    |
| `Любое` из *двух* выражений<br>`E1` или `E2`                       | `E1\|E2` для *grep*<br>и *sed*, `E1/E2` для *awk* | `one\|two` = или one, или two<br>`one\|two` = или one, или two                                                        |
| *Группировка* выражения<br>`E` с учетом *приоритета*               | `\(E\)` для *grep*<br>и `sed`, (E) для *awk*      | `\(one\|two)\*` = ноль или<br>более вхождений one или two,<br>`(one\|two)*` = ноль или более<br>вхождений one или two |

##  Регулярные  выражения на примере `grep`

> **Совет:**
> Всегда используем кавычки вокруг регулярных выражений в Bash, чтобы оболочка не пыталась интерпретировать *спецсимволы самостоятельно*.
