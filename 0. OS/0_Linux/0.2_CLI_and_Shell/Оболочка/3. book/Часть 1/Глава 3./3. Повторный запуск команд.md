### Повторный запуск команд

Предположим, только что выполнили длинную команду со сложным конвей-
ером,  `«Обнаружение дубликатов файлов»`

```bash
$ md5sum *.jpg | cut -c1-32 | sort | uniq -c | sort -nr
```
и вы хотите запустить ее еще раз. За кадром остается то, как оболочка
ведет запись вызываемых команд и их можно  повторнозапускать. Эта функция оболочки называется *историей команд.*

Точно так же предположим, что вы допустили ошибку при наборе предыдущей
команды, например написав `jg` вместо `jpg`:
```bash
$ md5sum *.jg | cut -c1-32 | sort | uniq -c | sort -nr
```
Оболочка поддерживает *редактирование командной строки* для исправления опечаток и выполнения всевозможных модификаций. Чтобы исправить ошибку, не нажимайте клавишу `Backspace` десятки раз и не вводите все заново.

`! Важно научиться печать в слепую. Попробуйте достичь 100 слов в минуту. Результат стоит потраченных усилий.`
