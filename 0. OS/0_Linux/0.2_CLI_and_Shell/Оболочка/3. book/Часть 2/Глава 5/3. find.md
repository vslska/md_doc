## `find`
Команда `find` *рекурсивно* выводит список файлов в каталоге. Cпускаясь по
подкаталогам и выводя полные пути.  Результаты выводяться не афавитном порядке.
```bash
$ find /etc -print      # Вывести список всех каталогов
```

Опции `find`
```bash
$  find . -type f -print # Только файлы
$  find . -type d -print # Только директории
$  find /etc -type f -name "*.conf" -print # Файлы, заканчивающиеся на .conf, шаблон заключаем в кавычки или экранируем (чтобы не вычислять)
$  find . -iname "*.txt" -print # Соответсвие имен шаблона не чувствительны к регистру. опция `-iname`
```

`find` также может выполнить команду Linux для всех файлов в выходных данных,
используя `-exec`. На примере  `ls  -l`
```bash
$  find /etc -type f -name "*.conf" -exec ls -l {} ";"
```
> Синтаксис выглядит несколько громоздким:
> 1. Набираем команду `find` без параметра `-print`.
> 2. Добавляем параметр `-exec`, а затем команду для выполнения. Используем скобки `{}`, чтобы указать, где в команде должен отображаться путь к файлу.
> 3. В конце выражения используем  точку с запятой в кавычках `";"` или экранированную `\;`.

`find -exec` можно использовать для *удаления файлов* во вложенных каталогах ( Острожно )
```bash
$  find $HOME/Pictures/Screenshots -type f -name "*.png" -exec echo rm {} ";"  # Используем echo для вывода списка удаляеммых
rm /home/vslsk/Pictures/Screenshots/snapshot_2026-01-01_01-37-25.png

$  find $HOME/Pictures/Screenshots -type f -name "*.png" -exec rm {} ";" # Удалить по-настоящему
```
