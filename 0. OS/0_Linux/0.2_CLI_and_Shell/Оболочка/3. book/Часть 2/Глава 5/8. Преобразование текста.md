## Преобразование текста
### 1. `tr`
Команда `tr` переводит один набор символов в другой.
```bash
$ echo $PATH | tr : "\n" # Преобразует двоеточие на символ новой строки
/home/vslsk/yandex-cloud/bin
/home/vslsk/bin
/home/vslsk/.local/bin
...
$ echo efficient | tr a-z A-Z # Перевод a в A, b в B и т. д.
EFFICIENT
$ echo Efficient | tr A-Z a-z
efficient

$ echo Efficient Linux | tr " " "\n" # Преобразование пробелов в символы новой строки
Efficient
Linux

$ echo efficient linux | tr -d ' \t' # Удаление пробелов (-d) и знаков табуляции '\t'
```

### 2. `rev`
Переворачивает символы задом наперед в каждой строке ввода
```bash
$ echo Efficient Linux! | rev
!xuniL tneiciffE
```

### 3. `awk` `sed`
Универсальные «суперкоманды» для обработки текста
```bash
$ sed 10q myfile       # Выводит 10 строк и завершается
$ awk 'FNR<=10' myfile # Выводит строки на экран, до тех пор пока их количество не достигнет 10
```
Позволяют заменять или менять местами строки:
```bash
$ echo image.jpg | sed 's/\.jpg/.png/'          # Заменить .jpg на .png
image.png
$ echo "linux efficient" | awk '{print $2, $1}' # Поменять местами два слова 		
efficient linux
```
