### `yes`
Команда `yes` выводит одну и ту же строку *снова и снова*, пока вы ее *не остановите*:
```bash
$  yes         # Выводит "y" (по умолчанию)
y
y
y ^C

$  yes goof!  # Вывести другую фразу
goof!
goof!
goof! ^C

```
`yes` может вводить данные для *интерактивных программ*, чтобы они могли работать без нашего вмешательства. Например, программа `fsck`, которая *проверяет* файловую систему *Linux* на наличие ошибок, может спросить пользователя о необходимости продолжения и будет ожидать ответа `y` или `n`.

Можно печатать строку  `n` раз 
```bash
$  yes "Efficient Linux" | head -n3 		# Вывести строку 3 раза
Efficient Linux
Efficient Linux
Efficient Linux
```
