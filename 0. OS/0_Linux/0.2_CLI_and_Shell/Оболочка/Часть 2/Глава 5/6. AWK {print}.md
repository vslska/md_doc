`AWK {print}`
Вывести `2е` слово в каждой строке.
```bash
$ awk '{print $2}' /etc/hosts # 0 вывести все столбцы
```


```bash
$ df / /data | awk '{print $4}'
$ df  / | awk '{print $0}' #  
Filesystem              1K-blocks     Used Available Use% Mounted on
/dev/mapper/ArchVG-root 405573632 61545568 339604032  16% / 
$ df  / | awk 'FNR>1 {print $0}' # Не выводить заголовок
/dev/mapper/ArchVG-root 405573632 61545956 339603676  16% /
```

Можно изменить  на другое регулярное выражение `-F` 
```bash
echo efficient:::::linux | awk -F':*' '{print $2}' # Любое количество двоеточий
linux  
```
