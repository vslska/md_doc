
###  `date`
Команда `date` выводит текущую *дату* и/или *время* в различных форматах.
```bash
$ date
Mon Jan 19 02:14:02 PM MSK 2026
$ date +%Y-%m-%d # Формат год - месяц - день
2026-01-19
$ date +%H:%M:%S # Формат часы : минуты : секунды
14:20:03
```

Чтобы управлять форматом вывода, укажим аргумент, начинающийся со знака
*плюс*, за которым следует любой текст. Текст может содержать специальные вы-
ражения, начинающиеся со знака процента, например *%Y* для четырехзначного
значения текущего года и *%H* для текущего часа в 24-часовом формате. Полный
список выражений находится на справочной странице команды `date`.

```bash
$ date +"Privet %A"
Privet Monday
```

### `seq`
Команда `seq`  печатает последовательность чисел из **диапозона**. Укажем **2 аргумента**, *нижнее и верхнее значение  диапозона*, и `seq` напечатает весь диапозон: 
```bash
seq 1 4  # Выводит все числа от 1 до 4
1
2
3
4
```

Если указать *3 аргумента*,  **1  и 3-й** определяют **диапозон значений**, **2-й** аргумент определяет *шаг*
```bash
$ seq 1 2 11 # Вывести диапозон с 2 до 11
1
3
5
7
9
11
```

Используйте **отрицательный** шаг, например  **-1**, для создания уменьшающейся последовательности:
```bash
$ seq 3 -1 0 # Вывеси диапозон с 3 до 0 с шагом -1 
3
2
1
0
```

Или  дробный шаг для получения чисел с плавающей запятой:
```bash
$ seq 1.1 0.1 1.8 # Вывести значения с плавающей  запятой с шагом 0.1
1.1
1.2
1.3
1.4
1.5
1.6
1.7
1.8
1.9
```

По умолчанию значения разделяются *символом новой строки*, но можно изменить разделитель с помощью параметра *-s*, после которого можно указать *любой символ*:
```bash
$ seq -s/ 1 5 # Вывести диапозон значений разделяя знаком /
1/2/3/4/5
```

Парметр *-w* приводит все значения к одинаковой ширине ( в символах ), добавляя *ведущие нули* по мере необходимости:
```bash
$ seq -w 8 12  # Вывести значения с одинаковой шириной
08
09
10
11
12
```

`seq` может выводить числа во многих других форматах.
