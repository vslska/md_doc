### Руководство по `awk`
**awk** — это не просто утилита, а целый язык программирования, предназначенный для обработки текстовых данных, представленных в виде строк и столбцов (таблиц, логов, конфигов).
#### 1. Базовый синтаксис
Программа на `awk` состоит из правил, которые следуют логике:
```bash
awk 'шаблон { действие }' файл
```
> - **Шаблон (Pattern):** Условие, при котором выполняется код. Если шаблон пуст, действие выполняется для каждой строки.
> - **Действие (Action):** Код в фигурных скобках `{...}`. Если действие пусто, `awk` просто выводит (печата) строку  # действие по умолчанию `{print}`

### 0.
> При вводе `awk-программы` в командной строке заключи ее в *кавычки*, чтобы оболочка не вычисляла специальные символы `awk`. При необходимости используй *одинарные* или *двойные кавычки*.
---
### 2. Работа со столбцами (Полями)

`awk` автоматически делит каждую строку на части (поля). По умолчанию разделитель — **пробел** или **табуляция**.
- `$0` — Вся строка целиком.
- `$1`, `$2`, `$3`... — Первый, второй, третий столбцы.
- `NF` — Встроенная переменная, означающая **количество полей** в текущей строке.
- `$NF` — Содержимое **последнего** поля.

**Пример:** Вывести только *фамилии и имена* (допустим, это 1-й и 2-й столбцы):
```bash
$ awk '{ print $1, $2 }' users.txt
Григорьев Тимофей
Вишневская Елизавета
Аксенова Таисия
```
---
### 3. Специальные блоки: BEGIN и END

Эти блоки выполняются вне цикла обработки строк.

|Блок|Когда выполняется|Типичное использование|
|---|---|---|
|**BEGIN**|До чтения первой строки|Настройка разделителей, печать заголовков.|
|**Тело**|Для каждой строки|Фильтрация, вычисления.|
|**END**|После обработки всех строк|Вывод итоговых сумм, средних значений.|
```bash
awk 'BEGIN { print "--- СТАРТ ---" } { print $1 } END { print "--- ГОТОВО ---" }' data.txt

```
---
### 4. Встроенные переменные

Знание этих переменных делает код в разы мощнее:

- **`FS`** (Field Separator): Разделитель полей на входе. По умолчанию — пробел.
- **`OFS`** (Output Field Separator): Разделитель полей на выходе (что ставится между запятыми в `print`).
- **`NR`** (Number of Record): Текущий порядковый номер строки во всем потоке.
- **`FNR`** (File Number of Record): Номер строки в **текущем** файле (сбрасывается для каждого нового файла).

**Пример изменения разделителя (для CSV-файлов):**
```bash
awk -F',' '{ print $1 }' data.csv
# Или через BEGIN
awk 'BEGIN { FS=":" } { print $1 }' /etc/passwd
```


### 5. Условия и регулярные выражения

`awk` поддерживает сложные фильтры:

- **Сравнение:** `==`, `!=`, `>`, `<`, `>=`, `<=`
- **Регулярки:** `~` (соответствует), `!~` (не соответствует)
- **Логика:** `&&` (И), `||` (ИЛИ)

**Примеры:**

1. Найти строки, где во втором столбце число больше 100:  
    `awk '$2 > 100' file.txt`
2. Найти строки, где первый столбец начинается на "Admin":  
    `awk '$1 ~ /^Admin/' file.txt`
3. Вывести строки с 10-й по 20-ю:  
    `awk 'NR >= 10 && NR <= 20' file.txt`

---

#### 6. Арифметика и переменные

`awk` умеет считать «на лету». Переменные не нужно объявлять.

**Пример: Подсчет суммы значений во втором столбце:**

```bash
awk '{ sum += $2 } END { print "Итоговая сумма:", sum }' sales.txt

```

### 7. Полезные "однострочники" (Cheat Sheet)

| Задача                                  | Команда                       |
| --------------------------------------- | ----------------------------- |
| Посчитать количество строк              | `awk 'END { print NR }' file` |
| Вывести только четные строки            | `awk 'NR % 2 == 0' file`      |
| Вывести последнюю колонку каждой строки | `awk '{ print $NF }' file`    |
| Найти строки длиннее 80 символов        | `awk 'length($0) > 80' file`  |
| Суммировать размер файлов в папке       | `ls -l`                       |

### Как запускать AWK-скрипты

Если программа стала слишком длинной, сохраю её в файл `script.awk`:

```bash
#!/usr/bin/awk -f
BEGIN { FS=":" }
$3 > 1000 { print "Пользователь-гигант: " $1 }

```

Запуск:
```bash
awk -f script.awk /etc/passwd
```
