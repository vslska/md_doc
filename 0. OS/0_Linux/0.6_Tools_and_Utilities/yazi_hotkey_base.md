## Yazi hotkey (base)

### Навигация
Для перемещения между файлами и каталогами можно использовать клавиши со стрелками. ←, ↓, ↑ и → или клавиши, похожие на клавиши Vim, например h, j, k, l:

| Назначение клавиш | Альтернативная тональность | Действие                                                                |
| ----------------- | -------------------------- | ----------------------------------------------------------------------- |
| `k`               | `↑`                        | Переместите курсор вверх                                                |
| `j`               | `↓`                        | Переместите курсор вниз.                                                |
| `l`               | `→`                        | Перейдите в каталог, на который был наведен курсор мыши.                |
| `h`               | `←`                        | Выйдите из текущей директории и перейдите в её родительскую директорию. |

Дополнительные команды навигации можно найти в таблице ниже.

| Назначение клавиш | Действие                                                                                                                                                                                                                |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `K`               | Найдите 5 единиц в предварительном просмотре.                                                                                                                                                                           |
| `J`               | Найдите 5 единиц в предварительном просмотре.                                                                                                                                                                           |
| `g`⇒ `g`          | Переместите курсор вверх                                                                                                                                                                                                |
| `G`               | Переместите курсор вниз.                                                                                                                                                                                                |
| `z`               | [Перейдите](https://yazi-rs.github.io/docs/configuration/keymap#mgr.cd) в каталог или [отобразите](https://yazi-rs.github.io/docs/configuration/keymap#mgr.reveal) файл с помощью fzf.                                  |
| `Z`               | [Скопируйте файл](https://yazi-rs.github.io/docs/configuration/keymap#mgr.cd) в каталог с помощью zoxide.                                                                                                               |
| g⇒ Space          | [Перейдите](https://yazi-rs.github.io/docs/configuration/keymap#mgr.cd) в нужную директорию или [отобразите](https://yazi-rs.github.io/docs/configuration/keymap#mgr.reveal) файл через интерактивную командную строку. |

### Выбор

Для выбора файлов и каталогов доступны следующие команды.

| Назначение клавиш | Действие                                                              |
| ----------------- | --------------------------------------------------------------------- |
| `Space`           | Переключить выделение файла/каталога, на который наведен курсор мыши. |
| `v`               | Перейдите в визуальный режим (режим выбора).                          |
| `V`               | Перейти в визуальный режим (режим отключения)                         |
| `Ctrl`+ `a`       | Выбрать все файлы                                                     |
| `Ctrl`+ `r`       | Обратный выбор всех файлов                                            |
| `Esc`             | Отменить выбор                                                        |

### Файловые операции

Для взаимодействия с выбранными файлами/каталогами используйте любую из команд ниже.

| Назначение клавиш | Действие                                                                                          |
| ----------------- | ------------------------------------------------------------------------------------------------- |
| `o`               | Открыть выбранные файлы                                                                           |
| `O`               | Открывайте выбранные файлы в интерактивном режиме                                                 |
| `Enter`           | Открыть выбранные файлы                                                                           |
| `Shift`+ `Enter`  | Открывайте выбранные файлы в интерактивном режиме (некоторые терминалы пока это не поддерживают). |
| `Tab`             | Показать информацию о файле                                                                       |
| `y`               | Скопировать выбранные файлы                                                                       |
| `x`               | Вырезать выбранные файлы                                                                          |
| `p`               | Вставьте скопированные файлы                                                                      |
| `P`               | Вставьте скопированные файлы (замените существующие файлы, если они уже есть в папке назначения). |
| `Y` или `X`       | Отменить статус "янк"                                                                             |
| `d`               | Удалить выбранные файлы                                                                           |
| `D`               | Безвозвратно удалить выбранные файлы                                                              |
| `a`               | Создайте файл (заканчивается на / для каталогов).                                                 |
| `r`               | Переименовать выбранный(е) файл(ы)                                                                |
| `.`               | Изменить видимость скрытых файлов                                                                 |

Дополнительные команды для работы с файлами можно найти в таблице ниже.

| Назначение клавиш | Действие                                                                   |
| ----------------- | -------------------------------------------------------------------------- |
| `;`               | Выполните команду оболочки                                                 |
| `:`               | Выполните команду оболочки (заблокируйте выполнение до завершения).        |
| `-`               | Символическая ссылка на абсолютный путь к загружаемым файлам               |
| `_`               | Символическая ссылка, указывающая относительный путь к загружаемым файлам. |
| `Ctrl`+ `-`       | Жесткие ссылки на скопированные файлы                                      |

### Копировать пути

Для копирования путей используйте любую из приведенных ниже команд.

_Наблюдение: c⇒ dуказывает на нажатие cсначала нажмите клавишу, а затем нажмите d ключ._

| Назначение клавиш | Действие                             |
| ----------------- | ------------------------------------ |
| `c`⇒ `c`          | Скопируйте путь к файлу.             |
| `c`⇒ `d`          | Скопируйте путь к каталогу.          |
| `c`⇒ `f`          | Скопируйте имя файла                 |
| `c`⇒ `n`          | Скопируйте имя файла без расширения. |

| Назначение клавиш | Действие                                                                               |
| ----------------- | -------------------------------------------------------------------------------------- |
| `s`               | Поиск файлов по имени с помощью [fd](https://github.com/sharkdp/fd)                    |

### Сортировка
Для сортировки файлов/каталогов используйте следующие команды.

_Наблюдение: ,⇒ aуказывает на нажатие ,сначала нажмите клавишу, а затем нажмите a ключ._

| Назначение клавиш | Действие                                               |
| ----------------- | ------------------------------------------------------ |
| `,`⇒ `m`          | Сортировать по времени изменения                       |
| `,`⇒ `M`          | Сортировка по измененному времени (в обратном порядке) |
| `,`⇒ `b`          | Сортировать по времени рождения                        |
| `,`⇒ `B`          | Сортировать по времени рождения (в обратном порядке)   |
| `,`⇒ `e`          | Сортировать по расширению файла                        |
| `,`⇒ `E`          | Сортировка по расширению файла (в обратном порядке)    |
| `,`⇒ `a`          | Сортировать по алфавиту                                |
| `,`⇒ `A`          | Сортировать в алфавитном порядке (в обратном порядке)  |
| `,`⇒ `n`          | Сортировать естественным образом                       |
| `,`⇒ `N`          | Сортировка естественным путем (в обратном порядке)     |
| `,`⇒ `s`          | Сортировать по размеру                                 |
| `,`⇒ `S`          | Сортировать по размеру (в обратном порядке)            |
| `,`⇒ `r`          | Сортировать случайным образом                          |

### Многовкладочный режим
| Назначение клавиш  | Действие                                      |
| ------------------ | --------------------------------------------- |
| `t`                | Создайте новую вкладку с CWD.                 |
| `1`, `2`, ..., `9` | Перейдите на вкладку N.                       |
| `[`                | Перейти на предыдущую вкладку                 |
| `]`                | Перейти на следующую вкладку                  |
| `{`                | Поменять местами текущую и предыдущую вкладку |
| `}`                | Поменять местами текущую и следующую вкладку  |
| `Ctrl`+ `c`        | Закрыть текущую вкладку                       |
