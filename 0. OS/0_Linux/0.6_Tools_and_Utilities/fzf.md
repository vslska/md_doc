### Синтаксис поиска
Если не указано иное, `fzf` запускается в «расширенном режиме поиска», в котором вы можете Введите несколько поисковых запросов, разделённых пробелами. Например: `^music .mp3$ sbtrkt !fire`

| Токен     | Тип совпадения                                        | Описание                                        |
| --------- | ----------------------------------------------------- | ----------------------------------------------- |
| `sbtrkt`  | нечеткое соответствие                                 | Соответствующие предметы `sbtrkt`               |
| `'wild`   | точное совпадение (цитата)                            | В число предметов входят `wild`                 |
| `'wild'`  | Точное совпадение границ (цитирование с обоих концов) | В число предметов входят `wild`на границах слов |
| `^music`  | префикс-точное совпадение                             | Предметы, начинающиеся с `music`                |
| `.mp3$`   | точное совпадение суффикса                            | Элементы, заканчивающиеся на `.mp3`             |
| `!fire`   | обратное точное совпадение                            | В список не входят `fire`                       |
| `!^music` | инверс-префикс-точное-совпадение                      | Элементы, которые не начинаются с `music`       |
| `!.mp3$`  | инверс-суффикс-точное-совпадение                      | Предметы, которые не заканчиваются на `.mp3`    |

Если не нравится нечеткое сопоставление и  не хотите «цитировать» каждое слово, запустить fzf с `-e` или `--exact`вариант. Обратите внимание, что когда `--exact`установлено, `'`-префикс "без кавычек" обозначает термин.
