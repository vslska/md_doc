Шпаргалка по Vim: Управление и Горячие Клавиши

Vim имеет несколько основных режимов работы. Переключение между ними — ключевой аспект эффективного использования редактора.

### 1. Нормальный режим (Normal Mode)

Это основной режим, в который вы попадаете при запуске Vim. Он используется для навигации, выполнения команд и переключения в другие режимы.

- **Переход в нормальный режим:** Нажмите клавишу `Esc`. Это ваша "домашняя" клавиша.

Навигация (Базовые движения)

| Клавиша           | Действие                                                              |
| ----------------- | --------------------------------------------------------------------- |
| `h`               | Переместить курсор влево                                              |
| `j`               | Переместить курсор вниз                                               |
| `k`               | Переместить курсор вверх                                              |
| `l`               | Переместить курсор вправо                                             |
| `w`               | Перейти к началу следующего слова                                     |
| `b`               | Перейти к началу предыдущего слова                                    |
| `0` (ноль)        | Перейти в начало текущей строки                                       |
| `$`               | Перейти в конец текущей строки                                        |
| `gg`              | Перейти в начало файла (первая строка)                                |
| `G`               | Перейти в конец файла (последняя строка)                              |
| `:<номер_строки>` | Перейти к конкретной строке (например, `:10` перейдет на 10-ю строку) |
| `Ctrl + f`        | Прокрутка страницы вниз (вперед)                                      |
| `Ctrl + b`        | Прокрутка страницы вверх (назад                                       |

Редактирование и удаление

| Клавиша     | Дейсвтвие                                                          |
| ----------- | ------------------------------------------------------------------ |
| `x`         | Удалить символ под курсором                                        |
| `dd`        | Удалить (вырезать) текущую строку целиком                          |
| `dw`        | Удалить от курсора до конца слова                                  |
| `d$`        | Удалить от курсора до конца строки                                 |
| `p`         | Вставить (paste) содержимое буфера обмена _после_ курсора/строки   |
| `P`         | Вставить (paste) содержимое буфера обмена _перед_ курсором/строкой |
| `yy`        | Скопировать (yank) текущую строку                                  |
| `yw`        | Скопировать слово от курсора                                       |
| `u`         | Отменить последнее действие (undo)                                 |
| `Ctrl + r`  | Повторить отмененное действие (redo)                               |
| `.` (точка) | Повторить последнее изменение/команду                              |

### 2. Режим вставки (Insert Mode)

Этот режим используется для непосредственного ввода текста, как в обычном текстовом редакторе.

- **Переход в режим вставки (из нормального режима):**
    - `i` (insert) — начать ввод _перед_ курсором.
    - `a` (append) — начать ввод _после_ курсора.
    - `o` — вставить новую строку _ниже_ текущей и начать ввод в ней.
    - `O` — вставить новую строку _выше_ текущей и начать ввод в ней.
    - `I` — перейти в начало строки и начать ввод.
    - `A` — перейти в конец строки и начать ввод.
- **Возврат в нормальный режим:** Нажмите `Esc`.

---

### 3. Визуальный режим (Visual Mode)

Этот режим используется для выделения блоков текста перед выполнением над ними команд (удаление, копирование, изменение).

- **Переход в визуальный режим (из нормального режима):**
    - `v` — начать посимвольное выделение.
    - `V` (заглавная V) — начать построчное выделение.
    - `Ctrl + v` — начать блочное (прямоугольное) выделение.
- **Действия после выделения:**
    - Используйте `h`/`j`/`k`/`l` или другие команды навигации для расширения выделения.
    - `d` или `x` — удалить выделенный текст.
    - `y` — скопировать (yank) выделенный текст.
- **Возврат в нормальный режим:** Нажмите `Esc`.

---

### 4. Режим командной строки (Command-Line / Ex Mode)

Этот режим используется для выполнения административных задач, таких как сохранение файла, выход из редактора, поиск и замена. Вы входите в него, набирая `:` (двоеточие) в нормальном режиме.

| Комманда             | Действиев                                                                                   |
| -------------------- | ------------------------------------------------------------------------------------------- |
| `:w`                 | Сохранить (write) файл                                                                      |
| `:q`                 | Выйти (quit), только если нет несохраненных изменений                                       |
| `:wq` или `:x`       | Сохранить и выйти                                                                           |
| `:q!`                | Выйти без сохранения изменений (игнорировать изменения)                                     |
| `:e <путь/к/файлу>`  | Открыть другой файл                                                                         |
| `/шаблон`            | Начать поиск _вперед_ по файлу (нажмите `n` для следующего совпадения, `N` для предыдущего) |
| `?шаблон`            | Начать поиск _назад_ по файлу                                                               |
| `:%s/старый/новый/g` | Найти все вхождения "старый" и заменить на "новый" во всем файле (`%`)                      |
