#### Что такое Jinja2?

Это **шаблонизатор** — движок, который берёт HTML-файл с _«дырками»_ и _логикой_, вставляет в них данные из Python — и отдаёт готовый HTML браузеру.
### Основные конструкции:
| Конструкция                                 | Назначение                       | Пример                                  |
| ------------------------------------------- | -------------------------------- | --------------------------------------- |
| `{{ переменная }}`                          | Подстановка значения             | `{{ name }}` → `"i.technology_devil"`   |
| `{% if условие %} ... {% endif %}`          | Условия                          | `{% if theme == 'lavender' %}`          |
| `{% for item in items %} ... {% endfor %}`  | Циклы                            | `{% for project in projects %}`         |
| `{% block имя %} ... {% endblock %}`        | Блоки (наследование)             | `{% block content %}`                   |
| `{% include 'файл.html' %}`                 | Вставка фрагмента                | `{% include 'partials/_header.html' %}` |
| `{% macro имя(парам) %} ... {% endmacro %}` | Переиспользуемые функции-шаблоны | `{% macro btn(text, href) %}`           |
### Как это работает в Flask:

1. Ты вызываешь `render_template('index.html', name="i.")`.
2. Flask ищет `index.html` в `./templates/`.
3. Jinja2 **парсит** шаблон: заменяет `{{ name }}` → `"i."`, выполняет `{% if %}`, `{% for %}` и т.д.
4. На выходе — чистый HTML, без следов Jinja.
5. Flask отправляет этот HTML в браузер.

> **Важно**: Jinja2 работает **на сервере**, до отправки в браузер. В клиенте — только HTML/CSS/JS.
