## Вариант 1

Моя шпаргалка команд n.1

| Задача                  | Команда                                                |
| ----------------------- | ------------------------------------------------------ |
| **Логи проекта**        | `kubectl logs -l app=my-web -f`                        |
| **Логи Nginx**          | `kubectl logs -l app.kubernetes.io/name=ingress-nginx` |
| **Проверка ресурсов**   | `kubectl get all`                                      |
| **Статус всех систем.** | `kubectl get pods -A`                                  |
| **Проверка дисков**     | `kubectl get pvc`                                      |
| **Проверка маршрутов**  | `kubectl get ingress`                                  |
| **Зайти внутрь пода**   | `kubectl exec -it [pod_name] -- /bin/bash`             |
| **Перезапуск аппки**    | `kubectl rollout restart deployment flask-personal`    |
| **Cписок HELM**         | `helm list`                                            |
