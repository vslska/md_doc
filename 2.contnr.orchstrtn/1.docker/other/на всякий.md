Правильный (Docker Network)

Контейнеры должны быть в одной виртуальной сети, чтобы видеть друг друга по именам.

1. **Создать сеть:**
```bash
docker network create my_app_net
```

2. **Подключить базу к этой сети** (если контейнер базы уже запущен с именем `pg_db`):
```bash
docker network connect my_app_net pg_db
```

3. **Запустить приложение в этой же сети**, изменив `localhost` в конфиге/коде на `pg_db`:
```bash
# В приложении DATABASE_URL должен быть postgresql://user:pass@pg_db:5432/db
docker run --name simple_app_v2 --network my_app_net -p 8000:8000 simple_python_app:v2
```
