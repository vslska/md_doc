## emasc hotkeys
**Обозначения:**
- `C-` — клавиша **Ctrl**
- `M-` — клавиша **Meta** (обычно это **Alt**)
- `S-` — клавиша **Shift**

---
### Файлы и буферы
| Сочетание | Действие                      |
| --------- | ----------------------------- |
| `C-x C-f` | **Открыть** файл (найти файл) |
| `C-x C-s` | **Сохранить** текущий файл    |
| `C-x C-w` | Сохранить как...              |
| `C-x b`   | Переключиться между буферами  |
| `C-x k`   | Закрыть (убить) буфер         |
| `C-x C-c` | **Выход** из Emacs            |

---
### Навигация
| Сочетание     | Действие                                |
| ------------- | --------------------------------------- |
| `C-p` / `C-n` | Вверх / Вниз (Previous/Next)            |
| `C-b` / `C-f` | На символ Назад / Вперед (Back/Forward) |
| `M-b` / `M-f` | На слово Назад / Вперед                 |
| `C-a` / `C-e` | В начало / конец строки                 |
| `M-a` / `M-e` | В начало / конец предложения            |
| `M-<` / `M->` | В самый **начало / конец** файла        |
| `C-v` / `M-v` | Страница вниз / Страница вверх          |
| `M-g g`       | Перейти на строку с номером...          |

---
### Редактирование
| Сочетание | Действие                                     |
| --------- | -------------------------------------------- |
| `C-SPC`   | Установить метку (начать выделение)          |
| `C-w`     | **Вырезать** (Kill) выделенное               |
| `M-w`     | **Копировать** выделенное                    |
| `C-y`     | **Вставить** (Yank)                          |
| `M-y`     | Циклическая вставка из истории буфера обмена |
| `C-k`     | Вырезать текст от курсора до конца строки    |
| `C-/`     | **Отмена** (Undo)                            |
| `C-g`     | Сбросить текущую команду / выделение         |

---
### Поиск и замена
| Сочетание | Действие                                |
| --------- | --------------------------------------- |
| `C-s`     | Поиск вперед (Incremental search)       |
| `C-r`     | Поиск назад                             |
| `M-%`     | Найти и **заменить** (с подтверждением) |
| `ENTER`   | Выход из поиска (остаться на месте)     |

---
### Работа с окнами

| Сочетание | Действие                             |
| --------- | ------------------------------------ |
| `C-x 2`   | Разделить горизонтально              |
| `C-x 3`   | Разделить вертикально                |
| `C-x o`   | Перейти в другое окно (Other)        |
| `C-x 0`   | Закрыть текущее окно                 |
| `C-x 1`   | Закрыть **все** окна, кроме текущего |

---
### Помощь и команды
| Сочетание | Действие                      |
| --------- | ----------------------------- |
| `M-x`     | Выполнить команду по названию |
| `C-h t`   | Встроенный учебник (Tutorial) |
| `C-h k`   | Описание функции клавиши      |
| `C-h f`   | Описание функции (команды)    |
