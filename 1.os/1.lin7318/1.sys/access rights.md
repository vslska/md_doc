# Cистема прав доступа Unix: OCT / BIN / MASK
## Таблица преобразования прав
| Символ        | Значение   | Бинарный | Восьмеричный (Oct) | Маска (бит) |
| ------------- | ---------- | -------- | ------------------ | ----------- |
| `r` (read)    | Чтение     | `100`    | `4`                | `1<<2` (4)  |
| `w` (write)   | Запись     | `010`    | `2`                | `1<<1` (2)  |
| `x` (execute) | Выполнение | `001`    | `1`                | `1<<0` (1)  |
| `-`           | Нет права  | `000`    | `0`                | `0`         |
## Как рассчитать восьмеричное значение:

```bash
Права для:   Owner (user) | Group | Others
Символы:        rwx       |  r-x  |  r--
Бинарные:      111       |  101  |  100
Сумма бит:   4+2+1=7    | 4+0+1=5 | 4+0+0=4
Результат:              →  0754
```
## Практическая таблица часто используемых прав

| Символьное  | Восьмеричное | Бинарное      | Типичное применение                     | Комментарий                   |
| ----------- | ------------ | ------------- | --------------------------------------- | ----------------------------- |
| `rwxrwxrwx` | `0777`       | `111 111 111` | Временные каталоги                      | Опасно: все могут всё         |
| `rwxr-xr-x` | `0755`       | `111 101 101` | Исполняемые скрипты, публичные каталоги | Стандарт для бинарников       |
| `rwxr-x---` | `0750`       | `111 101 000` | Сервисные каталоги                      | Доступ владельцу и группе     |
| `rw-r--r--` | `0644`       | `110 100 100` | Конфиги, текстовые файлы                | Читают все, пишет владелец    |
| `rw-r-----` | `0640`       | `110 100 000` | Конфиденциальные конфиги                | Только владелец+группа        |
| `rw-------` | `0600`       | `110 000 000` | Приватные ключи, пароли                 | Только владелец               |
| `rwx------` | `0700`       | `111 000 000` | Личные директории                       | Полные права только владельцу |

### Как читать и записывать права:
```bash
# Символьный режим (человекочитаемый)
u = user (владелец), g = group, o = others, a = all
+ = добавить право, - = убрать, = = установить точно

Примеры:
u+rwx    # Дать владельцу все права
g+rx     # Добавить группе чтение и выполнение
o-w      # Убрать запись у остальных
a=rX     # Все: чтение + выполнение (только если уже есть выполнение или это каталог)

# Восьмеричный режим (точный)
0755 = владелец: rwx (7), группа: r-x (5), остальные: r-x (5)
0640 = владелец: rw- (6), группа: r-- (4), остальные: --- (0)

# В Ansible можно использовать оба формата:
mode: '0755'           # Восьмеричный (рекомендуется)
mode: 'u+rwx,g+rx,o=rx' # Символьный (Ansible 2.8+)
```

> [!TIP] Быстрая проверка прав на хосте
>```bash
> # Посмотреть права в символьном и восьмеричном формате
>stat -c "%a %A %n" /etc/ssh/sshd_config
># Вывод: 600 -rw------- /etc/ssh/sshd_config
>```
