### Структура Каталогов

| Каталог      | Назначение                                                                                                           | Примеры статей / Темы                                                                                                                                                            |
| ------------ | -------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `1.sys/`     | Базовые концепции и основы работы Linux-системы + zаметки по управлению системой, сервисами, сетями и оборудованием. | Устройство файловой системы (FHS), процесс загрузки, основы прав доступа.Управление сервисами systemd, `nmcli`, `iproute2`, настройка фаерволов, **OOM Killer**, работа с ядром. |
