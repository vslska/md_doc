## 1.
### 1. Работа с разделителями (FS) 

Самая частая проблема: я вижу колонки, но `awk` считывает всё как одно поле `$1`.

#### Типы разделителей:

1. **По умолчанию (Пробелы/Табы):** `awk '{print $2}'` — делит по любому количеству пробелов. **Минус:** если в названии книги есть пробел (`Programming Python`), он разобьет его на два поля.
2. **Табуляция:** `awk -F'\t' '{print $2}'`. Работает только если в файле стоят реальные символы `Tab`.
3. **Регулярное выражение (Два и более пробелов):** Идеально для таблиц, которые выровнены пробелами для красоты.

```bash
awk -F'  +' '{print $2}' animals.txt
```

> _Здесь разделитель — это `+` (два и более пробела подряд)._

#### Как проверить, что внутри файла?

Если `awk` не видит колонки, проверю невидимые символы командой:

```bash
cat -A animals.txt
```

>- `^I` — это табуляция.
>- `$` — конец строки.
>- Если между словами просто пустота — это обычные пробелы.
---

### 2. Переменные и Поля

- `$0` — Вся строка.
- `$1, $2, ...` — Столбцы.
- `NF` — Количество столбцов в строке.
- `NR` — Номер текущей строки.

---

### 4. Специальные блоки: BEGIN и END

```awk
awk '
BEGIN { print "Начало отчета" }
$3 > 2010 { print $1, "выпущена после 2010" }
END { print "Всего строк обработано: " NR }
' animals.txt
```

---

### 5. Практические примеры для  данных

Если мой файл `animals.txt` выглядит так (*разделен пробелами*):  
`python Programming Python 2010 Lutz, Mark`

| Задача                             | Команда                                         |
| ---------------------------------- | ----------------------------------------------- |
| **Вывести только названия книг**   | `awk -F' +' '{print $2}' animals.txt`           |
| **Вывести животных и год**         | `awk -F' +' '{print $1, $3}' animals.txt`       |
| **Найти книги, изданные до 2005**  | `awk -F' +' '$3 < 2005 {print $2}' animals.txt` |
| **Посчитать кол-во книг в списке** | `awk 'END {print NR}' animals.txt`              |

---

### 6. Условия и Регулярки

- **Соответствие:** `~`
    - `awk '$1 ~ /cat/ {print $0}'` — вывести все строки, где первое поле содержит "cat".
- **Логические связки:**
    - `awk '$3 > 2000 && $3 < 2010' file` — книги между 2000 и 2010 годами.

---

### 7. Обработка CSV и других форматов

Если данные разделены запятыми или двоеточиями:

```bash
awk -F, '{print $1}' data.csv
awk -F: '{print $1}' /etc/passwd
```

### Совет по отладке

Если команда не работает, первым делом попробуйте напечатать количество найденных полей в каждой строке:
```bash
awk -F'\t' '{print NF}' animals.txt
```

>Если везде выводит `1`, значит `awk` не нашел ни одной табуляции и считает, что колонка всего одна. В этом случае меняйте `-F'\t'` на `-F' +'` или удаляйте флаг `-F` вовсе.
