# Создание файлов через EOF (Heredoc)(bash, zsh)

Метод **Heredoc** позволяет записывать многострочный текст в файл прямо из терминала или скрипта. Это особенно удобно для создания конфигов или быстрых заметок.

## 1. Базовый синтаксис
Для создания нового файла (или перезаписи старого):
```bash
cat << EOF > note.md
# Заголовок
Ваш текст здесь.
EOF
```

## 2. Добавление в конец файла
Если нужно дописать текст в уже существующий файл, используйте `>>`:
```bash
cat << EOF >> noe.md
Это новая строка в конце файла.
EOF
```

## 3. Отключение обработки переменных
Если в тексте есть спецсимволы (например, `$`), которые не нужно превращать в переменные окружения, оберните `EOF` в кавычки:
```bash
cat << "EOF" > script.sh
echo "Переменная $USER не будет раскрыта"
EOF
```

## 4. Полезные фишки
- **Ctrl + D**: Если вы просто ввели `cat > file.txt`, это сочетание клавиш отправит сигнал конца файла (EOF) и сохранит изменения.
- **cat <<- EOF**: Если добавить дефис, Bash будет игнорировать табуляцию в начале строк (удобно для красивого кода внутри скриптов).

## 5. Быстрый шаблон для Obsidian
```bash
cat << EOF > "$(date +%Y-%m-%d).md"
---
date: $(date +%Y-%m-%d)
---
# Новая заметка
- [ ] Сделать: 
EOF
```
