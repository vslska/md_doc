### Получение и обработка HTML из командной строки
В Linux можно **загружать, парсить и извлекать данные из веб-страниц** без браузера — с помощью `curl`, `wget`, текстовых браузеров и специализированных утилит.

#### 1. Загрузка страниц: `curl` vs `wget`
| Утилита | По умолчанию       | Когда использовать                           |
| ------- | ------------------ | -------------------------------------------- |
| `curl`  | выводит в `stdout` | для конвейеров, скриптов, API                |
| `wget`  | сохраняет в файл   | для загрузки файлов, рекурсивного скачивания |
Примеры:
```bash
# Просто получить содержимое
curl https://efficientlinux.com/welcome.html

# Сохранить как welcome.html
wget https://efficientlinux.com/welcome.html
```

 **Маскировка под браузер** (если сайт блокирует скрипты):
```bash
curl -A "Mozilla/5.0" URL
wget -U "Mozilla/5.0" URL
```

#### 2. Массовая загрузка файлов

**Способ 1: через `xargs` (рекомендуется)**
```bash
seq 1 20 | xargs -I@ wget https://efficientlinux.com/images/@.jpg
```

**Способ 2: через `awk` + `bash`**
```bash
seq 1 20 | awk '{print "wget https://.../" $1 ".jpg"}' | bash
```

> **Почему `xargs` лучше?**  
 Он не интерпретирует спецсимволы (`$`, `*`, `;`) — безопаснее при  непредсказуемых URL.

#### 3. Извлечение структурированных данных из HTML

Если страница содержит таблицу или список — можно **извлечь данные в TSV/CSV**.
##### Инструменты:

- `curl` — загрузка
- `hxnormalize` — нормализация HTML
- `hxselect` — выбор элементов по CSS-селекторам (из пакета `html-xml-utils`)
##### Пример: извлечь телефонные коды
```bash,
curl -s https://efficientlinux.com/areacodes.html \
| hxnormalize -x \
| hxselect -c -s'@' '#ac .ac, #ac .state, #ac .cities' \
| sed 's/\([0-9]*\)@\([A-Z]\{2\}\)@\([^@]*\)@/\1\t\2\t\3\n/g'
```
→ Результат:
```bash
201	NJ	Hackensack, Jersey City
202	DC	Washington
...
```
**Читаемость регулярных выражений**:  
Разбивайте сложные `sed`-выражения на переменные:
```bash
areacode='\([0-9]*\)'
state='\([A-Z]\{2\}\)'
cities='\([^@]*\)'
sed "s/$areacode@$state@$cities@/\1\t\2\t\3\n/g"
```

#### 4. Текстовые браузеры: `lynx`, `links`

Когда нужно **читаемое представление**, а не сырой HTML:
```bash
lynx -dump https://efficientlinux.com/areacodes.html > table.txt
```
 → Вывод уже похож на таблицу — удобно для быстрого анализа.

> Плюсы:
> - Не выполняет JavaScript → безопаснее
> - Легко парсится через `awk`/`grep`

#### 5. Работа с буфером обмена из терминала

В Linux два буфера:

- **Primary selection** — выделение текста мышью (вставка — средняя кнопка)
- **Clipboard** — явное копирование (`Ctrl+Shift+C`)

##### Утилиты для CLI:

- `xclip` (чаще в Debian/Ubuntu)
- `xsel` (чаще в Fedora)

**Примеры**:
```bash
# Скопировать вывод в буфер
echo "secret" | xclip -selection clipboard

# Вставить из буфера
xclip -selection clipboard -o

# Автоочистка через 15 сек
(echo "temp" | xclip -sel c; sleep 15; echo -n | xclip -sel c) &
```

> **Безопасность**:  
Никогда не копируйте пароли в буфер надолго — используйте автоочистку.

### Итог

Вы научились:

- **загружать** веб-контент без браузера,
- **извлекать структурированные данные** из HTML,
- **массово скачивать** файлы,
- **работать с буфером обмена** из терминала.

Эти навыки позволяют:

- автоматизировать сбор данных,
- интегрировать веб-API в скрипты,
- безопасно анализировать подозрительные ссылки.

> Главный принцип:  
> **Веб — это тоже текст. А текст — универсальный формат для обработки в Unix.**
