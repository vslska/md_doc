## Объединение текста
#### 1.  `cat` (concatenates)
Может содержимое нескольких файлов, последовательно объединяя сверху-вниз
```bash
$ cat poem1
It is an ancient Mariner,
And he stoppeth one of three.
$ cat poem2
By thy long grey beard and glittering eye,
$ cat poem3
Now wherefore stopp st thou me?
$ cat poem1 poem2 poem3
It is an ancient Mariner,
And he stoppeth one of three.
By thy long grey beard and glittering eye,
Now wherefore stopp'st thou me?
```
---
#### 2.  `tac` 
Построчно переворачивает каждый текст
```bash
$ cat poem1 poem2 poem3 | tac
Now wherefore stopp'st thou me?
'By thy long grey beard and glittering eye,
And he stoppeth one of three.
It is an ancient Mariner,
```
---
####  3. `paste`
Объединяет строки текстов в столбцы, разделенные одним символом табуляции.
```bash
$ cat title-words1
EFFICIENT
AT
COMMAND

$ cat title-words2
linux
the
line

$ paste title-words1 title-words2
EFFICIENT  linux
AT the
COMMAND line

$ paste -d, title-words1 title-words2 # -d (delimiter) изменить разделитель на другой символ
EFFICIENT,linux
AT,the
COMMAND,line

$ paste -d, -s title-words1 title-words2 # Изменить вывод поменяв последовательность -s 
EFFICIENT,AT,COMMAND
linux,the,line

$ paste -d "\n" title-words1 title-words2 # 
EFFICIENT 
linux
AT
the
COMMAND
line
```
---
#### 4. `diff`
Cравнивает два файла построчно и выводит краткий отчет об их различиях:
```bash
$  cat file1
Linux is all about efficiency.
I hope you will enjoy this book.
$  cat file2
MacOS is all about efficiency.
I hope you will enjoy this book.
Have a nice day.
$  diff file1 file2
1c1   # код ## 1я строка отличается в срав. файлах
< Linux is all about efficiency.
---   # Разделитель 
> MacOS is all about efficiency.
2a3   # код ## во 2м есть 3я строка 
> Have a nice day.
```

Можно вывести разные строки с помощью `diff`, `grep` и `cut`:
```bash
$  diff file1 file2 | grep '^[<>]'
< Linux is all about efficiency.
> MacOS is all about efficiency.
> Have a nice day.
$  diff file1 file2 | grep '^[<>]' | cut -c3-
Linux is all about efficiency.
MacOS is all about efficiency.
Have a nice day.
```
